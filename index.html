Some Recent Examples of Projects I Have Worked On

Finesse Fitness

A full-stack single page app using React in the front end and Node + Express in the backend to allow users to view over 1300 exercises and create custom workouts from them. Exercises are stored in a noSQL mongoose database and the website is deployed on Heroku.

View Demo Visit GitHub Repo

Yard Barker

A full-stack Django app that connects job posters with workers to get work done around the house. Users can post a job and have it viewed by other users on the app. Jobs are location based and the google maps API is leveraged to display that data. All data is stored in a PostgreSQL database. Created with Max Wang, RusLan Didun, and Andy Krauss

View Demo Visit GitHub Repo

Tourney of Champs

Full-stack app that allows users to register a team and sign players up to that team. Built using EJS in the front end and an express + node.js backend. Data is stored in a mongoDB database
